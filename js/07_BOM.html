BOM (Browser Object Model)

window

    window 객체는 자바스크립트의 최상위 객체이며, 브라우저 내의 모든 요소들과 객체를 포함한다.
    BOM : location 객체, navigator 객체, history 객체, screen 객체
    DOM : document 객체

    window.open();
    window.setTimeout();
    window.setInterval();

location

    location.reload();
    location.href;
    location.assign;
    location.replace();

history

    history.back();
    history.forward();
    history.go(-1);
    history.go(1);

navigator

    브라우저 정보를 담고있는 객체

screen

    화면(모니터) 정보를 담고있는 객체
