기본문법

1. 변수와 자료형

    var     : 중복선언 O, 재할당 O, hoisting 발생 | 사용X  
    let     : 중복선언 X, 재할당 O, 
    const   : 중복선언 X, 재할당 X, 

    * var, let, const 키워드 없이 그냥 만들변 전역변수 
    ex> x1 = '전역변수'; 
    
    자바스크립트에서는 변수에 별도로 자료형을 지정 X, 리터럴에 의해 자료형이 결정
  • string(문자열)
  • number(숫자)
  • boolean(논리값)
  • object(객체)
  • function(함수)
  • undefined(초기화가 안된 변수)

2. 데이터 형변환

    자바스크립트에서 문자형 -> 숫자형 변환 시
    Number 내장 객체 또는 parseInt(), parseFloat() 내장 함수를 이용하여 강제 형변환 가능
    자바스크립트에서 숫자형 -> 문자형 변환 시
    String 내장 객체를 이용하여 강제 형변환이 가능하다.

3. 연산자

    == | !=  : 내용만 비교
    === | !== : 타입까지 비교
